How it works

The complete 7-step workflow ‚Äî from session launch to verified close.

1

Launch / connect session

Start a managed browser, connect via CDP, or relay through the extension.

opendevbrowser launch --no-extension
2

Navigate & snapshot

Load your target URL and capture a deterministic DOM snapshot.

opendevbrowser goto ‚Üí opendevbrowser snapshot
3

Act via stable refs

Click, type, select, press, scroll ‚Äî using stable element references.

opendevbrowser click "ref:submit-btn"
4

Observe & debug

Poll console, network, trace snapshots, and perf metrics in real time.

opendevbrowser console-poll ‚Üí opendevbrowser perf
5

Run workflows

Execute packaged research, shopping, or product video pipelines.

opendevbrowser research run --topic "query"
6

Export & capture

Screenshot, clone pages/components, annotate ‚Äî export in any format.

opendevbrowser clone-page
7

Verify & close

Confirm session status, take follow-up snapshots, disconnect cleanly.

opendevbrowser status ‚Üí opendevbrowser disconnect
